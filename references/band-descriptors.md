# 雅思写作官方评分标准详解

基于 IELTS Public Band Descriptors，用于精确评分参考。

---

## Task 1 评分标准

### Task Achievement (TA) — 任务完成度

| 分数 | 学术类 Task 1 | 培训类 Task 1 |
|------|-------------|-------------|
| **9** | 完全满足任务要求；清晰呈现完整展开的概述；巧妙突出并阐述所有关键特征/数据 | 完全满足任务要求；语气一致且恰当；清晰呈现完整展开的概述；所有要点充分展开 |
| **8** | 覆盖所有任务要求；清晰呈现并恰当展开概述；适当突出并阐述关键特征/数据 | 覆盖所有任务要求；语气一致且恰当；清晰呈现概述并恰当展开；所有要点充分展开 |
| **7** | 覆盖任务要求；清晰呈现概述；突出并阐述了关键特征/数据，但部分展开不充分 | 覆盖任务要求；语气基本一致恰当；有清晰概述；所有要点得到展开，但部分不够均衡 |
| **6** | 回应任务要求但部分覆盖不完整；概述不够清晰；关键特征/数据有涉及但展开不充分或包含不相关细节 | 回应任务要求但语气可能不一致；概述不够清晰；要点有呈现但展开不充分 |
| **5** | 基本回应任务但可能格式不当；概述不清晰或缺失；机械罗列数据无清晰概述；关键特征/数据有遗漏 | 基本回应任务但格式可能不当；语气不一致；概述不清晰；部分要点遗漏或展开不充分 |
| **4** | 仅回应部分任务要求；格式可能不当；概述缺失；数据/特征主要依靠照搬题目。字数可能不足 | 仅回应部分任务要求；格式不当；语气不当；要点遗漏严重。字数可能严重不足 |
| **3** | 几乎未回应任务；没有概述；数据/特征有严重误读 | 几乎未回应任务；格式严重不当；要点几乎未呈现 |
| **2** | 回复与任务几乎无关 | 回复与任务几乎无关 |
| **1** | 回复完全无关 | 回复完全无关 |
| **0** | 未作答 / 完全背诵 / 非英语 | 同左 |

### Coherence & Cohesion (CC) — 连贯与衔接

| 分数 | 描述 |
|------|------|
| **9** | 无缝衔接；分段自如；衔接手段运用精妙 |
| **8** | 信息与观点逻辑推进顺畅；全文衔接自然；分段合理；每个段落有清晰的中心主题 |
| **7** | 信息与观点逻辑推进；衔接手段运用恰当；有清晰的中心主题推进；分段合理但可能有些机械 |
| **6** | 信息与观点推进基本连贯；衔接手段运用有效但不总是流畅；分段基本合理；有使用指代和替换但可能有重复 |
| **5** | 信息组织有一定逻辑但整体推进不够流畅；衔接手段不足或使用不准确/过度；缺少指代和替换导致重复；分段不足或不当 |
| **4** | 信息观点不连贯；衔接手段使用有限且不准确；分段不足或缺失 |
| **3** | 观点之间无逻辑联系；衔接手段极少且几乎不准确 |
| **2** | 几乎无组织 |
| **1** | 完全无组织 |
| **0** | 未作答 / 完全背诵 / 非英语 |

### Lexical Resource (LR) — 词汇资源

| 分数 | 描述 |
|------|------|
| **9** | 词汇极其丰富，运用自如精准；罕见小错误且仅属笔误；巧妙使用非常见词汇和地道表达 |
| **8** | 词汇丰富，流畅传达精确含义；恰当使用非常见词汇和地道表达；偶尔有选词和搭配不准确；拼写/构词错误极少 |
| **7** | 词汇资源充足，可灵活运用；使用部分非常见词汇并注意风格和搭配；偶有选词不当；拼写/构词错误少且不影响理解 |
| **6** | 词汇资源足够完成任务；尝试使用非常见词汇但有时不准确；拼写/构词有错误但不影响理解 |
| **5** | 词汇资源有限，仅能应对任务基本需求；选词明显错误导致理解困难；拼写/构词错误频繁；经常重复用词 |
| **4** | 词汇资源极为有限，仅能完成最基本任务；词汇使用严重受限；拼写/构词频繁出错严重影响理解 |
| **3** | 词汇极端有限；几乎无法表达意思 |
| **2** | 仅有个别孤立词汇 |
| **1** | 无有效词汇 |
| **0** | 未作答 / 完全背诵 / 非英语 |

---

### Grammatical Range & Accuracy (GRA) — 语法范围与准确性

| 分数 | 描述 |
|------|------|
| **9** | 句型丰富多样，运用自如精准；语法极少出错且仅属笔误 |
| **8** | 句型丰富多样；大部分语句无语法错误；偶有小错但极少出现系统性问题 |
| **7** | 句型多样；能写出无语法错误的句子；但部分句子可能有语法问题 |
| **6** | 使用简单句和复合句混合；复杂句尝试但准确度不如简单句；语法错误时有但不严重影响理解；标点基本正确 |
| **5** | 以简单句为主；复杂句尝试有限且经常出错；语法错误频繁导致部分理解困难；标点错误常见 |
| **4** | 句型极为有限；复杂句极少且几乎全部错误；语法错误频繁且严重妨碍理解 |
| **3** | 几乎没有有效句子；语法错误极其严重 |
| **2** | 仅有孤立词组；几乎无法构成句子 |
| **1** | 无语法的单词堆砌 |
| **0** | 未作答 / 完全背诵 / 非英语 |

## Task 2 评分标准

### Task Response (TR) — 任务回应

| 分数 | 描述 |
|------|------|
| **9** | 全面回应所有问题；立场清晰且贯穿全文；观点充分展开，论证有力；论据相关且充分支撑 |
| **8** | 回应所有问题；立场清晰且一致；观点展开充分；论据相关且有说服力 |
| **7** | 回应所有问题；立场清晰一致；主要论点展开充分，但部分可以更深入；论据相关 |
| **6** | 回应所有问题但部分回应不够充分；有立场但结论可能不清晰或重复；主要论点有展开但不够充分或有重复；论据基本相关但不够有力 |
| **5** | 仅回应部分问题；立场可能不清晰；论点展开有限，论证不充分；论据可能不相关或不足 |
| **4** | 仅最低限度回应任务；立场不明确；论点极少展开或完全不展开；论据不相关或缺失 |
| **3** | 几乎未回应任务；无清晰立场 |
| **2** | 回复与任务几乎无关 |
| **1** | 回复完全无关 |
| **0** | 未作答 / 完全背诵 / 非英语 |

## 评分实操指南

### 6 分 vs 7 分的关键区别

**Task 2 TR 6 → 7 的跳跃：**
- 6 分：回应问题但展开不够充分，"has some good points but could be more fully extended"
- 7 分：所有问题都清楚回应，主要论点展开充分，立场贯穿全文

**LR 6 → 7 的跳跃：**
- 6 分：尝试使用不常见词汇但有时不准确
- 7 分：灵活使用不常见词汇，注意风格和搭配，错误少

**GRA 6 → 7 的跳跃：**
- 6 分：简单句与复合句混合，复杂句尝试但准确度不高；虽出错但大意可懂
- 7 分：句型多样，大部分句子无语法错误

### 中国考生典型分数段

| 分数段 | 占比 | 典型特征 |
|-------|------|---------|
| 5.0-5.5 | ~30% | 模板痕迹重，论证浅，语法错误频繁，词汇重复 |
| 5.5-6.0 | ~35% | 有模板但部分展开，简单句为主，词汇够用不丰富 |
| 6.0-6.5 | ~20% | 结构合理，论证有展开，词汇有变化但不够地道，语法偶错 |
| 6.5-7.0 | ~10% | 逻辑清晰，论证充分，词汇较丰富，语法错误少 |
| 7.0+ | ~5% | 观点深入，语言地道，句型多样，几乎无语法错误 |
